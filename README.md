# pdd-184 电商实时销售大屏

## 简介
全屏电商大促实时销售数据可视化大屏，模拟电商大促场景，展示销售额、订单数、品类排行、全国热力图、订单状态及每分钟订单量。

## 技术栈
- React 18 + TypeScript + Vite
- TailwindCSS
- ECharts (echarts-for-react)
- lucide-react

## 安装与运行

```bash
npm install
npm run dev
```

打开浏览器访问 `http://localhost:3002`。

## 功能
- 顶部滚动：最新成交信息（用户 + 商品 + 金额）
- 左侧：今日销售额和订单数数字大屏，数字带增长动画
- 左侧下方：品类销售排行榜（横向条形图）
- 中间：中国地图热力图，各省颜色深浅表示销售额，点击省份查看详情
- 右侧：订单状态分布环形图（待付款/待发货/已发货/已完成）
- 右侧下方：近一小时每分钟订单量折线图
- 数据每 4 秒自动刷新模拟实时效果
